# clickymeow
React memory game.
[Link to page](https://reekamaharaj.github.io/clickymeow/)

## Description
Clicky Meow is a memory game built with React. Click on a different cat to increase your score!

![](clickymeow.png)

## How to Use
Click the pictures, but don't click the same one twice!
